# Definition of Done

> A task is not done until every item below is true. "It works on my machine" is not done.

---

## Code
- [ ] Feature works as specified
- [ ] No console errors in the browser
- [ ] No broken links or missing assets
- [ ] Code is readable — would a future-you understand it in 3 months?

## Database
- [ ] Migration file created and committed
- [ ] RLS policies verified for the affected table(s)
- [ ] Change tested against real data (not just schema validation)

## Documentation
- [ ] PROGRESS.md updated with milestone entry
- [ ] SESSION_HANDOFF.md reflects current state
- [ ] Any new architectural decision documented (why, not just what)

## Deploy
- [ ] Committed and pushed to correct branch
- [ ] GitHub Pages live URL reflects the change
- [ ] No regressions on existing features

## GitHub
- [ ] Related Issue closed with commit reference
- [ ] Project card (`projects/[name].md`) updated
